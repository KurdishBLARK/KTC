

ئەركی جێناوی نادیار لە ڕستەدا

ئەم ڕستانە بخوێنەوە:
1- فڵان زۆرزانە.
2- كەس لە ئاهەنگی نەورۆز دوانەكەوت.
3- هوین هەمێی دزانن.
4- ئەوان هەواڵەكەیان بە فڵان گوتووە.
5- هەسپێ‌ كابرای هاتە دزین.
6- هین بۆ دەرەوەی وڵات نێرراوە.
7- كەم و كورتیەكە لە كابرا بوو.
8- مافی كەس مەخۆ.

خستنەڕوو: 

پێشتر زانیمان، كە جێناوی نادیار لە بڕی ناوێك لە ڕستەدا بەكاردێت، لەبەر ئەمە هەموو ئەركەكانی ناو لە ڕستەدا دەبینێت وەكو :
1- لە ڕستەی یەكەمدا وشەی (فڵان) جێناوی نادیارە ئەركی نیهادی بینیوە.
2- لە ڕستەی دووەمدا پەیڤی (كەس) جێناوی نادیارە ، ئەركی بكەری بینیوە.
3- لە ڕستەی سێیەمدا پەیڤی (هەمیێ‌) جێناوی نادیارە ئەركی بەركارە لە ڕستەدا بینیوە.
4- لە ڕستەی چوارەمدا وشەی (فڵان) جێناوی نادیارە ئەركی تەواوكەری بە یاریدەیەی لە ڕستەدا بینیوە.
5- لە ڕستەی پێنجەمدا پەیڤی (كابرا) جێناوی نادیارە ئەركی تەواوكەری ناوە لە ڕستەدا ،ناوی (هەسپی) پێ‌ تەواوكراوە واتە دیارخەری ناوە.
6- لە ڕستەی شەشەمدا وشەی (هین) جێناوی نادیارە، ئەركی جێگری بكەری بینیوە.
7- لە ڕستەی حەوتەمدا وشەی (كابرا) جێناوی نادیارە ئەركی تەواوكەری بەیاریدەی كاری ناتەواوەی لە ڕستەدا بینیوە.
8- لە ڕستەی هەشتەمدا وشەی (كەس) جێماوی نادیارە ئەركی دیارخەری ناوی (ماف)ی بینیوە لە ڕستەدا. 

دەستور:
جێناوی نادیار، كە لە بڕی ناوێك بەكاردێت ، ئەركی ناو لە ڕستەدا دەبینێت. وەكو (بكەر، نیهاد، بەركار، تەواوكەری بەیاریدە، دیارخەری ناو، تەواوكەری كاری ناتەواو، جێگری بكەر).


ڕاهێنان ( 1 )
ئەم جێناوانەی خوارەوە لەڕستەدا بەكاربهێنە:
 گشت، هەندێك، كەس، چ، هەرامە، وانەك.
  

ڕاهێنان ( 2 )
جێناوی (فڵان) لەڕستەدا بەكار بهێنە بەمەرجێك هەموو ئەركەكانی ناو ببینێت.
ڕاهێنان ( 3 )
لەم ڕستانەی خوارەوەدا جێناوە نادیارەكان دەربهێنە و ئەركەكانیان دیاربخە. 
1- خڕ هاتبوون.
2- مە چ نەكریە.
3- تۆ هینت بۆكوێ‌ نارد؟
4- یەكێك لە هاوڕێكانم زۆر زیرەكە.
5- ئەم هەمێی دزانین. 
6- ئەمە فڵان بوو.
7- بلا كەس نەهێتە گرتن. 

ڕاهێنان ( 4 )
ئەم ڕستانەی خوارەوە شلۆڤە بكە:
1- یەكێ‌ لە كتێبەكانم بە كابرا بەخشی.
2- شڤانی چ نەنڤیسییە.

ڕاهێنان ( 5 )
ئـ- جیاوازی لەنێوان هاوەڵناوی نادیار و جێناوی نادیار چییە؟ بەنمونە ڕوونیبكەوە. 

ب- ئەم پەیڤانە جارێك بەهاوەڵناوی نادیارو جارێكیش بە جێناوی نادیار بەكاربێنە:
 (هیچ، هەمی، گشت، چ، هین)
