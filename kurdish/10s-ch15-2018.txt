  شوشەكە شكا                     ڕێبوار شووشەكەی شكاند

  گایەكە بۆڕانــدی                 گایەكەی هێنایە بۆڕاندن  





خستنەڕوو: 
 
 كاتێك سەیری ڕستەكانی لای دەستە ڕاستی بەشی ژمارە (2 ، 1) دەكەین، دەبینین كاری (گەیشت، نووست، بەزی، شكا، رووخا، ترسا، لەدەست چوو، ناڵاند، باڕاند، بۆڕاند) لە ڕستەدا بەكار هێنراون، هەر یەكێك لەم كارانە لە شوێنی خۆیدا كارێكی تێنەپەڕە ،چونكە كاری كەسێك یان شتێكی دەرخستووە لە كاتێكی دیاری كراودا ، هەروەها كاریگەری كارەكە لە بكەر تێنەپەڕیوە،واتە تەنها بكەری وەرگرتووە و مانای تەواوی بەخشیوە. 
بەڵام كاتێك سەیری ڕستەكانی لای دەستە چەپی بەشی ژمارە        (1-2) دەكەین، دەبینین كاری (گەیاند، نوواند، بەزاند، شكاند، ڕوخاند، ترساند، لەدەستچوواند، هێنایە ناڵاندن، هێنایە باڕاندن، هێنایە بۆڕاندن) لە ڕستەكەدا بەكارهێنراون، هەریەكێك لەم كارانە لە شوێنی خۆی دا كارێكی تێپەڕە، چونكە كاریگەرییەكەی لە بكەر تێپەڕیوە و كەوتۆتە بۆ بەركار، واتە بێجگە لە بكەر پێویستی بە بەركار هەیە تا واتای تەواو ببەخشێت.
ئینجا لە كاتی گۆڕینی كاری تێنەپەڕ بۆكاری تێپەر پێویستیمان بەڕەگی كار یا قەدی چاوگ هەیە، كە پاشگری (اندن) دەخەینە سەر ڕەگی كار یان قەدی چاوگ، چاوگێكی تێپەر دێتە كایەوە، دەتوانرێت لە چاوگە تازەكە كاری ڕابردوو، ڕانەبردوو، داخوازی دروستبكرێن بەپێی ئەو دەستوورانەی، كە پێشتر خوێندوومانن، چاوگی تێنەپەڕ بەم شێوەیە دەگۆڕین بۆچاوگی تێپەڕ:

1- ئـ  
چاوگی تێنەپەڕ             ڕەگی كار              پاشگر          چاوگی تێپەڕ
گەیشتن                  گە  +  یـ            اندن         گەیاندن 
نووستن                 نــــــوو               اندن          نوواندن 
بەزین                    بــــــەز               اندن         بەزاندن 
مردن                    مـــر                 اندن          مراندن
ڕۆیشتن                 ڕۆ                   اندن         رواندن 
تەپین                    تەپ                 اندن          تەپاندن  

ب:
چاوگی تێنەپەڕ             قەدی چاوگ           پاشگر      چاوگی تێپەڕ                       
شكان                    شكا                 اندن      شكاندن 
ڕووخان                 ڕووخا               اندن     ڕووخاندن 
ترسان                   ترسا                اندن      ترساندن  
گریان                    گریا                 اندن      گریاندن 
لەدەست چوون         لەدەست چوو       اندن      لەدەست چوواندن 

2- هەندێك چاوگی تێنەپەڕ هەن كە لە دەنگە سروشتییەكان دروستكراون بەیارمەتی پاشگری (اندن) وەكو (ناڵاندن، باڕاندن، بۆڕاندن، قیژاندن، نووزاندن......) ئەم چاوگانە هەر چەندە لە دەنگی سروشتیاكان و پاشگری (اندن) دروست كراون بەڵام هەر تێنەپەڕن، واتە بەپێی ئەو دەستوورەی پێشتر بەهۆی (اندن) ناكرێن بە چاوگی تێپەڕ، بەڵكو دەتوانرێت بەیارمەتیی كاری یاریدەدەری (هێنایە) یان (دەهێنێتە) كە دەخرێتە پێش چاوگی تێنەپەڕ بكرێن بە چاوگی تێپەڕ بەم شێوەیە : 
دەنگی سروشتی     پاشگر     چاوگی تێنەپەڕ      كاری یاریدەر     چاوگی تێپەڕ                                      
   ناڵە             اندن       ناڵاندن         هێنایە         هێنایە ناڵاندن
   باڕە            اندن       باڕاندن        هێنایە          هێنایە باڕاندن
   بۆڕە           اندن        بۆڕاندن       هێنایە         هێنایە بۆڕاندن
   قیژە            اندن       قیژاندن        هێنایە         هێنایە قیژاندن
                                               دەهێنێتە      دەهێنێتە قیژاندن

ڕوونكردنەوە:

ئەگەر ڕەگی كار یا قەدی چاوگ یا دەنگی سروشتی بە پیتی بزوێن كۆتایی هاتبوو، پیتێكی نەبزوێنی بۆ زیاد دەكەین یان پیتە بزوێنەكە لادەبەین. چونكە دوو پیتی بزوێن بەدوای یەكتردا نایەن.




دەستور:
گۆڕینی كاری تێنەپەڕ بۆ كاری تێپەڕ:

ئـ- (ڕەگی كار + اندن) =چاوگی تێپەڕ
ب- (قەدی چاوگ + اندن) = چاوگی تێپەڕ
پ- (دەنگە سروشتی + اندن) = چاوگی تێنەپەڕ
(كاری یاریدەدەر هێنایە + چاوگی تێنەپەڕ) = چاوگی تێنەپەڕ.

ڕاهێنان ( 1 )

چۆن كاری تێنەپەڕو كاری تێپەڕ لێك جیادەكەیتەوە؟ بە نمونە ڕوونیبكەرەوە.

ڕاهێنان ( 2 )

بەچەند ڕێگا كاری تێنەپەڕ دەگۆڕێت بۆ كاری تێپەڕ؟ بە نمونە ڕوونیبكەرەوە.

ڕاهێنان ( 3 )

چۆن لە دەنگە سروشتییەكان كاری تێپەڕ دروست دەكرێت؟                          بە چەند هەنگاو؟  بە نمونە ڕونیبكەرەوە.

ڕاهێنان ( 4 )

ئەم كارە تێنەپەڕانەی خوارەوە بگۆڕە بۆ كاری تێپەڕ و لە ڕستەدا بەكاریان بهێنە.
(تێپەڕی، خشی، كەوت، برژا، خنكا، سووتا، وەری، چوو، ڕەڤی، هەڕمی)

ڕاهێنان ( 5 )

ئەم ڕستانەی خوارەوە شلۆڤە بكە.
1- پێشمەرگەی كوردستان لەشكری دوژمنی بەزاند.
2- كارەساتەكە مرۆڤەكەی هێنایە قیژاندن.
