

كاری بكەر دیارو بكەر نادیار
لە شێوەزاری كرمانجی خواروودا
       ( ئـ )                              ( ب )
   كاری بكەر دیار               كاری بكەر نادیار
1. شاد گوڵەكەی چاند.                     1. گوڵەكە چێنرا.
2. كوڕەكان وانەكانیان نووسییەوە.          2. وانەكان نووسرانەوە.
3. جوتیارەكە زەوییەكە دەكێڵێت.           3. زەوییەكە دەكێڵرێت.
4. تاڤگە نامەكانی دەخوێنێتەوە.           4. نامەكان دەخوێنرێنەوە.
	
	لەمەوبەر زانیمان، كاری تێپەڕ، بەهۆی وشەیەكی ترەوە واتای ڕستەكە تەواودەكات‌و دەبێتە بەركاری كارەكە، ئەم ڕاستییە لە ڕستەكانی كۆمەڵەی (ئـ) دا بە تەواوی بەدیدەكرێت.
1. لە ڕستەی یەكەمدا (شاد) كە (بكەر)ە كارێكی كردووەو ئەركی كارەكەی یەكسەر كەوتۆتە سەر (گوڵەكە)و بووە بە بەركاری كاری (چاند)و واتای ڕستەكەشی تەواوكردووە، بەڵام لە ڕستەی یەكەمی بەشی (ب)دا لەبەر ئەوەی نەمانزانیوە كێ‌ (گوڵەكەی چاند)ووە، یان نەمانویستووە ناوی (شاد) بێنین، كە (بكەر)ی كارەكەیە، ڕستەكەمان لە شێوەی ڕستەی بكەر نادیار دەربڕیوە. 
	بۆ ئەم مەبەستەش، (بكەر)ی كارە ئاشكرایەكەو هەرچی وشەیەك، كە پەیوەندی بە (بكەر)ەوە هەیە لە ناو ڕستەكەدا لای دەبەین‌و بەركارەكە دەچێتە شوێنی‌و كارەكە دەخەینە سەر شێوەی كارێكی بكەر نادیار، بەو شێوەیەی كە لەسەرەوە پیشانمان داوە، بەم پێیە ڕستەكە دەبێتە (گوڵەكە چێنرا)و (گوڵەكە) دەبێتە جێگری (بكەر).
2. لە ڕستەی دووەمدا هەمان ڕێگەمان گرتووە بۆ گۆڕینی كاری بكەر دیار بۆ كارێكی بكەر نادیار، بەڵام كە (بەركار)ەكە دەچێتە شوێنی (بكەر) پێویستە بە پێی كەس‌و ژمارەی جێگری (بكەر)ەكە كارە بكەر نادیارەكەش بگۆڕێت، لەبەر ئەمە ڕستەكە دەبێتە (وانەكان نووسرانەوە).
3. لە ڕستەكانی (سێیەم‌و چوارەم)دا كاری (دەكێڵێت)و (دەخوێنێتەوە) دوو كاری ڕانەبردووی تێپەڕن، چونكە (بكەر)ەكانیان ئاشكران، بۆ گۆڕینی ئەم جۆرە كارانە، بە پێی ئەو دەستورەی لە سەرەوە دەرمانخست، (بكەر)ەكان لەناو ڕستەكاندا لا دەبەین‌و (بەركار)ەكان دەخەینە شوێنیان‌و كارەكانیش لەو شێوانەی، كە لە ڕستەكانی كۆمەڵەی (ب)دا نیشاندراون بەكار دەهێنین.
	نیشانەی ( رێ‌ ) كە بۆ كاری ڕانەبردووە هەروەها جێناوی لكاوی (ێت) دەچێتە سەر (رێ‌) دەبێتە (رێت)

 

دەستور:
1. كاری بكەر دیار: ئەو كارەیە كە یەكسەر دەخرێتە پاڵ (بكەر)ەكەی‌و لە ناو ڕستەكەشدا باس دەكرێت.
2. كاری بكەر نادیار: ئەو كارەیە كە (بكەر)ەكەی لەناو ڕستەدا باس ناكرێت‌و (بەركار)ەكەی دەبێتە جێگری (بكەر)ەكەی.
3. تەنیا كاری تێپەڕی دیار دەگۆڕێت بۆ كاری بكەر نادیار.
4. لەم گۆڕینەدا پێویستە بە وردی سەرنجی (تاك‌و كۆ)ی جێگری (بكەر) بدەین‌و بەو پێیە ئاگاداری ژمارەو كەسی كارە بكەر نادیارەكە بین.
5. ئەگەر كارە تێپەڕەكە:
ئـ - ڕابردوو بوو، ڕەگی كار وەردەگرین‌و پاشگری (را)ی دەخەینە سەر وەك:
چاوگ	ڕەگی كار	 ڕابردووی نادیار
 خووساندن	خووسێن	خوسێن + را = خوسێنرا
كێڵان  	 كێڵ	  كێڵ + را = كێڵرا
 شوشتن	 شۆ	  شۆ + را = شۆرا
نووسین	نووس	نووس + را = نووسرا

                       
ب‌-  ڕانەبردوو بوو، ڕەگی كار وەردەگرین‌و پاشگری (رێ‌)ی دەخەینە سەر. وەك:
چاوگ	ڕەگی كار	 رانەبردووی نادیار
 نووسین	 نووس	 دە+ نووس + ڕێ‌ = دەنووسرێ
خوێندن	 خوێن	 دە + خوێن + ڕێ‌ = دەخوێنرێ
 ‌تێبینی: 
1. كاتێك هەندێ‌ كار دەكرێن بە كاری بكەر نادیار لە دەستور لادەدەن، چونكە بە ڕەگی كار ناكرێن بە كاری بكەر نادیار، بەڵكو بە قەدی چاوگ دەكرێن، واتە:
ئـ . ڕابردوو
 قەدی چاوگ + را  
 گوتن: گوت + را = گوترا
 ویستن: ویست+ را = ویسترا
ب. ڕانەبردوو
 دە + قەدی چاوگ + رێ
 دە + گوت + رێ‌ = دەگوترێ‌
 دە + ویست+ رێ‌ = دەویسترێ‌
2. لە كاتی دروستكردنی كاری بكەر نادیار، ئەگەر هاتوو ڕەگی كار كۆتایی بە بزوێن هات، ئەوا پەیڕەو‌ی ئەم خاڵا‌نە دەكەین
ئـ . بزوێنی (ێ، ە) لە ناو دەچن. 
 دانان         دا + نێ + را‌           دانرا
 كردن        كە + را                 كرا
ب. بزوێنی (ۆ) دەگۆڕێت بۆ بزوێنی ( و)
  خواردن       خۆ + را       خورا
3. كاری بكەر نادیار بۆ كاتی ڕابردوو هەموو دەمكاتەكانی هەیە وەك (نزیك، تەواو، بەردەوام، دوور)



ڕاهێنانی (1)
	ئەم ڕستانە لە كاری بكەر دیارەوە بگۆڕە بۆ كاری بكەر نادیار:
1. منداڵەكە هێلكەیەكی خوارد.
2. قارەمانێك فڕۆكەیەكی دوژمنی خستە خوارەوە.
3. ئەندازیارەكان خانووەكان دروستدەكەن.
4. ئەوان چوار جار وانەكە دەنووسنەوە.
5. ئەو هەموو ڕۆژێ‌ سێ‌ جار دەم‌وچاوی دەشوات.

ڕاهێنانی (2)
	ئەم كارانە لە كاری بكەر نادیارەوە بگۆڕە بۆ كاری بكەر دیار:
1. ڕۆمانەكە چاپكرا.
2. دوو چاڵ هەڵكەنرا.
3. دوێنێ‌ دیوارەكان‌و سەربانەكان سواخدران.
4. شەش مانگە باخەكە ئاو دەدرێ.
5. تابلۆیەك دەكێشرێت.

ڕاهێنانی  (3)
	ئەم كارە بكەر نادیارانە لە ڕستەدا بەكاربهێنە:
(خورا، دەنوسرێت، دەبینرێت، دەخوێنرێ، پێشكەشدەكرێ، هەڵدرا).



ڕاهێنانی  (4)
	وەڵامی ڕاست هەڵبژێرە:
1. پەنجەرەكە شكێنرا، (شكێنرا) كاری ڕابردووی بكەر نادیاری:
    (نزیك/ دوور/ تەواو/ بەردەوام)
2. نیشانەی كاری بكەر نادیار لە كرمانجی خواروو بۆ ڕابردوو:
  ( رێ  /    را   /    هاتە)
3. گوڵەكە چێنرابوو، (چێنرابوو) كاری رابردووی بكەر نادیاری:
    (نزیك،  دوور،  بەردەوام، تەواو)
4. دیواری باخەكە ڕوخێنرا، ئەو ڕستەیە بكەر:
    (دیاره  /  نادیاره  /   هیچیان)
5. كاری دیار لە ڕستەدا یەكسەر دەخرێتە پاڵ: 
    (بكەر/  هاوەڵناو/  ئامرازی دانەپاڵ) 

ڕاهێنانی (5)
	شیكردنەوە:
1. دیواری باخەكە دروستكرا.
دیوار   : ناوە، تاكە، نەناسراوە، دیارخرا‌وە.
ی  : ئامرازی دانەپاڵ.
باخەكە: ناوە، تاكە، ناسراوە، دیارخەری ناوی (دیوار)ە.
دیواری باخەكە: گرێی ناوییە، جێگری بكەرە.
دروستكرا: كاری ڕابردووی نزیكی بكەر نادیارە.
2. ئەم ڕستانە شیبكەوە:
ئـ - تۆپەكە هێنرابوو.
ب-  پاقلەكە لە مەنجەڵەكەدا كوڵێنراوە.
پ - گەنم چۆن دەچێنرێت؟
