
كاری بكەر دیارو كاری بكەر نادیار
لەشێوەزاری كرمانجی سەروودا

           (ئـ)                                        (ب)
     كاری بكەر دیار                   كاری بكەر نادیار
1. شڤانی پەز چەراند.                    1. پەز هاتە چەراندن.
2. ڕەزڤانی ڕەزێ‌ خۆ ئاڤدایە.            2. ڕەز هاتییە ئاڤدان.
3. وێ‌ نان دخوار.                        3. نان دهاتە خوارن.
4. وی كەوەك گرتبوو.                   4. كەوەك هاتبوو گرتن.
5. پەری چیرۆكێ‌ دنڤیسیت.              5. چیرۆك دهێتە نڤیسین.
6. ئەز نامەكێ‌ دێ‌ نڤیسم.                6. نامەك دێ‌ هێتە نڤیسین.

	كاتێك سەیری ڕستەكانی بەشی (ئـ) دەكەین، دەبینین پەیڤەكانی (چەراند، ئاڤدایە، دخوار، گرتبوو، دنڤیسیت، دێ‌ نڤیسم) هەموویان كاری تێپەڕن، چونكە كاریگەری كارەكانی كەوتۆتە سەر ناوێكی تر كە (بەركار)ەو، هەر یەكەیان كردەوەیەكیان پیشانداوەو خراونەتە پاڵ بكەرەكانیان‌و هەموویان بەسەر یەكەوە ڕستەیەكی بكەر دیاریان پێكهێناوە.
    كاتێك سەیری ڕستەكانی بەشی (ب) دەكەین، دەبینین پەیڤەكانی (هاتە چەراندن، هاتییە ئاڤدان، دهاتە خوارن، هاتبوو گرتن، دهێتە نڤیسین، دێ‌ هێتە نڤیسین) هەموویان كاری بكەر نادیارن، چونكە هەر یەكەیان كردەوەیەكی پیشانداوەو نەخراونەتە پاڵ كەسێكی دیارو ئاشكرا (واتە نازانرێ‌ كێ‌ كارەكەی ئەنجامداوە)، لەبەر ئەوە بەم جۆرە كارانە دەگوترێت كاری بكەر نادیار.
	كەواتە لە دوای خوێندنەوەو تێڕامانمان لە شێوەزاری (كرمانجی سەروو) ئەوەمان بۆ دەردەكەوێت، كە یاسای گۆڕینی كاری بكەر دیار بۆ كاری بكەرنادیار جیاوازە لە شێوەزاری (كرمانجی خواروو)، چونكە لە كرمانجی سەروودا كاری ڕستەكە دەگەڕێتەوە بۆ (چاوگ)و پاشان كاری یاریدەدەری (هات)یش بە پێی جۆری كاتی كارەكە بەكار دەهێنین.
 
كاری ڕابردوو:
1. كاری ڕابردووی نزیك:
	ئەم جۆرە كارە بە پێی ئەم یاسایە دروست دەكرێت:
هاتە + چاوگ
هاتە + چاندن  هاتە چاندن
هاتە + كڕین    هاتە كڕین
2. كاری ڕابردووی تەواو:
	ئەم جۆرە كارەش بەپێی ئەم یاسایە دروست دەكرێت:
هاتییە + چاوگ
هاتییە+ نڤیسین   هاتییە نڤیسین
هاتییە + لێنان     هاتییە لێنان

3. كاری ڕابردووی بەردەوام:
دهاتە + چاوگ
دهاتە + ئینان =  دهاتە ئینان
دهاتە + ڕێشتن = دهاتە ڕێشتن
4. كاری ڕابردووی دوور:
هاتبوو+ چاوگ
هاتبوو + خوارن  = هاتبوو خوارن
هاتبوو + خواندن  = هاتبوو خواندن (خاندن)
 
كاری ڕانەبردوو:
	لە شێوەزاری كرمانجی سەروودا كاری ڕانەبردوو بۆ دەمی ئێستاو ئایندە بە ئاشكرا دەردەكەوێت لە كاتی ئاخاوتندا، لەبەرئەوە پێویستە ڕەچاوی ئەم خاڵانە بكرێت لە كاتی گۆڕینی كاری بكەردیار بۆ كاری بكەرنادیار:

1. كاری ڕانەبردوو بۆ دەمی ئێستا:
كاری یاریدەدەری (دهێتە) بەكاردێت لە پێش چاوگەكەدا:
دهێتە + چاوگ
دهێتە + كڕین  = دهێتە كڕین
دهێتە + نڤیسین = دهێتە نڤیسین



2. كاری ڕانەبردوو بۆ دەمی ئایندە:
كاری یاریدەدەری (دێ‌ هێتە) لە پێش چاوگەكەدا بەكاردێت:
دێ‌ هێتە + چاوگ
دێ‌ هێتە + فرۆتن  = دێ‌ هێتە فرۆتن
دێ‌ هێتە + هاڤێتن  = دێ‌ هێتە هاڤێتن

دەستور:
1. كاری بكەردیار: 
	ئەو كارەیە، كە كردەوەو ڕەفتارێك لە كاتێكی دیاریكراو (ڕابردوو یان ڕانەبردوو) دا پیشانداوە یان پیشاندەدات‌و ئەو كارەش، كە دراوەتە پاڵ بكەرەكەی، دیارو ئاشكرایە لە ڕستەكەدا.
2. كاری بكەرنادیار: 
	ئەو كارەیە، كە كردەوەو ڕەفتارێك لە كاتێكی دیاریكراو (ڕابردوو یان ڕانەبردوو)دا پیشانداوە یان پیشاندەدات‌و ئەو كارەش نەدراوەتە پاڵ بكەرەكەی‌و دیارو ئاشكرا نییە لە ڕستەكەدا.

ڕاهێنانی (1)
	لەم ڕستانەی خوارەوەدا كاری بكەر دیار بگۆڕە بۆ كاری بكەر نادیار:
1.سەرداری وانە نڤیسی.
2. وێ‌ هەڤیر كربوو.
3. نەسرین دێ‌ شیڤێ‌ لێنێت.
4. من كتێب ژ وی وەرگرتییە.

ڕاهێنانی (2)
	وەڵامی ڕاست هەڵبژێرە:
1. نیشانەی كاری بكەرنادیار لە كرمانجی سەروودا بۆڕانەبردوو دەمی ئێستا:
     (ڕەگی كار + رێ / دهێتە + چاوگ / دێ هبێتە + چاوگ)
2. نیشانەی كاری بكەر نادیار لە كرمانجی سەروودا بۆ ڕابردووی نزیك:
    (دهاتە + چاوگ /  دێ هێتە + چاوگ /  هاتە + چاوگ)
3. پەز هاتە چەراندن، (هاتە) كارێكی:
    (تێپەڕە / ڕابردووی تێنەپەڕە /  یاریدەدەرە)
4. كەوەك هاتییە گرتن، ڕستەیەكی بكەر نادیارە كارەكەی:
    (ڕابردووی تەواوە /  ڕانەبردووی ئایندە /  ڕانەبردووی ئێستا)
5. نان دهاتە خوارن، ڕستەیەكی بكەر نادیارە كارەكەی ڕابردووی:
    (تەواو /  نزیك /  بەردەوام)

ڕاهێنانی  (3)
	ئەم ڕستانەی خوارەوە شییبكەوە:
1. پەز دێ‌ هێتە دۆشین.
2. گۆڤارەكان بە چاكی دابەشكراون.
