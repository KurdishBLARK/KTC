
 (وانەى یازدەهەم)
هـاوەڵـناوى نـیشانە
ئـ-

1.	ئەم گوڵە جوانە.
2.	ئەو گوڵە جوانە.
3.	ئەم دارانە بەرزن.
4.	ئەو دارانە بەرزن.
5.	ئەڤ  كوڕەیێ زیرەكە.
6.	ڤى خورتى سەرێ مە بلندكر.
7.	ڤێ كچێ هۆزانەك ڤەهاند.
8.	وى جامێرى هاریكاریا مە كر.
9.	وێ گۆتنێ دلێ مە خۆشكر.
10.	ئەڤان مرۆڤان نان خار.
11.	وان جوتیاران زەڤیێن خۆ كێلان.

ب-
1.	ئەم دێیە گەورەیە.
2.	ئەو دێیانە گەورەن.
3.	ئەم كەیبانووە خاوێنە.
4.	ئەو كەیبانووانە خاوێنن.

 
خستنەڕوو:
ئـ- كاتێ سەیرى ڕستەكان دەكەین، دەبینین وشەكانى (ئەم...ە، ئەو... ە، ئەڤ ... ە ، ڤى ... ى، ڤێ ... ێ، وى... ى، 
وێ... ێ، ئەڤان... ان، وان ...ان) هەموویان هاوەڵناوى نیشانەن، چونكە هەریەكەیان وەسفى ناوێكى كردووە، بەهۆى دەستنیشانكردنەوە لە هاوچەشنەكانى خۆیدا جیایكردووەتەوە.

1.	لە ڕستەى یەكەمدا هاوەڵناوى نیشانەى (ئەم... ە) بۆ نزیك بەكاردێت، وشەى (گوڵ)ى دەستنیشانكردووە، وەكو سیفەتێك توانیویەتى لە هاوچەشنەكانى خۆیدا جیاى بكاتەوەو بووەتە دیارخەرى ناوى (گوڵ).
2.	لە ڕستەى دووەمدا هاوەڵناوى نیشانەى (ئەو...ە) بۆ دوور بەكاردێت، وشەى (گوڵ)ى دەستنیشانكردووە، وەكو سیفەتێك توانیویەتى لە هاوچەشنەكانى خۆیدا جیاى بكاتەوەو بووەتە دیارخەرى ناوى (گوڵ).
3.	لە ڕستەى سێیەمدا هاوەڵناوى نیشانەى (ئەم... ە ) بۆ نزیك بەكاردێت‌و وشەى (داران)ى دەستنیشانكردووە، وەكو سیفەتێك توانیویەتى لە هاوچەشنەكانى خۆیدا جیاى بكاتەوەو بووەتە دیارخەرى ناوى (داران).
4.	لە ڕستەى چوارەمدا هاوەڵناوى نیشانەى (ئەو... ە) بۆ دوور بەكاردێت‌و وشەى (داران)ى دەستنیشانكردووە، وەكو سیفەتێك توانیویەتى لە هاوچەشنەكانى خۆیدا جیاى بكاتەوەو بووەتە دیارخەرى ناوى (داران).
5.	لە ڕستەى پێنجەمدا هاوەڵناوى نیشانەى (ئەڤ ... ە ) بۆ نزیك بەكاردێت، وشەى (كوڕ)ى دەستنیشانكردووە، وەكو سیفەتێك توانیویەتى لە هاوچەشنەكانى خۆى جیاى بكاتەوەو بووەتە دیارخەرى ناوى (كوڕ).
6.	لە ڕستەى شەشەمدا هاوەڵناوى نیشانەى (ڤى... ى) بۆ ناوى نێرى تاكى نزیك بەكاردێت‌و وشەى (خورت)ى دەستنیشانكردووە، وەكو سیفەتێك توانیویەتى لە هاوچەشنەكانى خۆیدا جیاى بكاتەوەو بووەتە دیارخەرى ناوى (خورت).
7.	لە ڕستەى حەوتەمدا هاوەڵناوى نیشانەى (ڤێ... ێ) بۆ ناوى 
مێی تاكى نزیك بەكاردێت، وشەى (كچ)ى دەستنیشانكردووە، وەكو سیفەتێك توانیویەتى لە هاوچەشنەكانى خۆیدا جیاى بكاتەوەو بووەتە دیارخەرى ناوى (كچ).
8.	لە ڕستەى هەشتەمدا هاوەڵناوى نیشانەى (وى... ى) بۆ ناوى نێرى تاكى دوور بەكاردێت. وشەى (جامێر)ى دەستنیشانكردووە، وەكو سیفەتێك توانیویەتى لەناو هاوچەشنەكانى خۆیدا جیاى بكاتەوەو بووەتە دیارخەرى ناوى (جامێر).
9.	لە ڕستەى نۆیەمدا هاوەڵناوى نیشانەى (وێ... ێ ) بۆ ناوى مێی تاكى دوور بەكاردێت، وشەى (گۆتن)ى دەستنیشانكردووە، وەكو سیفەتێك توانیویەتى لەناو هاوچەشنەكانى خۆیدا، جیاى بكاتەوەو بووەتە دیارخەرى وشەى (گۆتن).
10.	لە ڕستەى دەیەمدا هاوەڵناوى نیشانەى (ئەڤان... ان ) بۆ ناوى نێرو مێى كۆى نزیك بەكاردێت، وشەى (مرۆڤ)ى دەستنیشانكردووە وەكو سیفەتێك توانیویەتى لە هاوچەشنەكانى خۆیدا جیاى بكاتەوەو بووەتە دیارخەرى ناوى (مرۆڤ).
11.	لە ڕستەى یازدەهەمدا هاوەڵناوى نیشانەى (وان... ان) بۆ ناوى نێرو مێ ى كۆى دوور بەكاردێت. وشەى (جوتیار)ى دەستنیشانكردووە، وەكو سیفەتێك توانیویەتى لەناو هاوچەشنەكانى خۆیدا جیاى بكاتەوەو بووەتە دیارخەرى ناوى (جوتیار).

ب-  لە ڕستەكانى (ب)دا وشەكانى (دێ، كەیبانوو) دوو ناون بە پیتى بزوێنى (ێ، وو ) كۆتاییان هاتووە، هەروەها هاوەڵناوى نیشانەكەش (ئەم... ە ) و (ئەو... ە ) بە پیتى بزوێنى (ە) كۆتایى هاتووە، بۆ ئەوەى دوو بزوێن بە دواى یەكدا نەیەن، ئەوا ناوبەندى گونجاو دەخەینە نێوانیانەوە، ئەو كاتە بەپێى مەبەست بەكاریان دێنین:
( ئەم + دێ +  یـ + ە )
( ئەو + كەیبانوو + و+ ە )


دەستور:
هاوەڵناوى نیشانە: ئەو هاوەڵناوەیە كە وەسفى ناوێك دەكات‌و بە ئاسانى لە هاوچەشنەكانى خۆى جیاى دەكاتەوەو دەبێتە دیارخەرى ناوەكە،  هەمیشە ناوە دەستنیشانكراوەكە دەكەوێتە نێوان هەردوو بەشى هاوەڵناوەكەوە.

هاوەڵناوەكانى نیشانە لە زمانى كوردیدا ئەمانەن:
1. ئەم .... ە	2. ئەو... ە	3. ئەڤ ... ە
4. ڤى... ى	5. ڤێ ... ێ	6. وى ... ى
7. وێ... ێ	8. ئەڤان... ان	9. وان ... ان

ڕاهێنانى (1)
نمونە:
1.	منداڵ (هەژار). ئەم منداڵە هەژارە پێویستى بە یارمەتى هەیە.
2.	دۆ (شیرین). ئەم دۆیە زۆر شیرینە.
ئەمانەى خوارەوە لەگەڵ هاوەڵناوى نیشانە لە ڕستەدا بەكاربهێنە:
{ دەرگا (پان)، پەنیر (بەتام)، ئاو (سارد)، دەریا (قووڵ)، ئاگر (خۆش)، درەخت (بەرز)، بەڕوو (شیرین)، پیاو (ئازا)، 
كەو (گەردن بە خاڵ)}

ڕاهێنانى (2)
لەم ڕستانەى خوارەوەدا هاوەڵناوى نیشانە دەربهێنە:
1.	ئەو گۆڤارانە بهێنە.
2.	ڤێ دەفتەرێ بكڕە.
3.	ئەم نەمامەم چاندووە.
4.	وى هەڤالى فراڤین خار.

ڕاهێنانى (3)


ئەم هاوەڵناوە نیشانانەى خوارەوە لە ڕستەدا بەكاربهێنە:
(وان... ان، ئەڤان ...ان، وێ... ێ، ئەم...ە، ئەو... ە )


ڕاهێنانى (4)
ئەم ڕستانەشلۆڤەبكە:
1.	ئەم خانووە خۆشە.
2.	ئەو وێنانە ڕەنگدەكەم.
3.	وێ كابانێ مالەكا مەزن بڕێڤەبر.
