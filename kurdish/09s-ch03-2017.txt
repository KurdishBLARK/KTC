

 چـــاوگ 
لە ڕووی پێكهاتنەوە 
یەكەم:
1. كوتانی منداڵان دژی نەخۆشییەكان پێویستە.
2. ب هاتنا وە ئەم دلشاد بووین.
3. وڵاتەكەمان بە چاندنی درەخت دەڕازێتەوە.
4. نرخی دروونی كراسەكە گران بوو.
5. پشكفتنا بشكۆژێن گوڵان ل بهاران دبیت.
6. خوێندنم تەواو كردووە.
دووەم: 
                  ( ئـ )
1. ڤەكرنا خوێندنگەهان وەلات پێشدكەڤیت.
2. هەڵبژاردنەكەیان ئەنجامدا.
3. لە ڕاسپاردنەكەت ‌گەیشتم.
4. تێگهشتن بناغا زانستێیە.
5. بە خوێندنەوە زانیاریی نوێ‌ فێردەبین.
6. قرتاندنی گوڵی باخان جوان نییە.
7. هەڵدانەوەی لاپەڕەیەكی نوێ‌ بۆ ئێمە پێویستە.
                (ب)
1. ئاڤدانێت باغچان گەلەك پێدڤییە.
2. ئێمە بۆ یارمەتیدانی ئێوە هاتووین.
3. جوانكردنی شارەكان ئەركێكی نیشتمانییە.
4. بە گەرمبوون تەنە ڕەقەكان لێكدەكشێن.
5. دەست پێ‌ كرنا هەر كارەكی پێویستی ب نەخشە دانانێ‌ هەیە.
6. ڕاستكردنەوەی هەڵەكانم بینی.
7. ل ناڤبرنا زۆرداریێ‌ ئارمانجا مەیە.
8. ڕێگەدانی چاپكردنی كتێبەكەم وەركرت.
9. بە بەهێزكردنی پەیوەندی نێوان لایەنەكان، كۆمەڵگە سەقامگیر دەبێت.
خستنەڕوو:
     لە بابەتی پێشوودا ئەوەمان زانی، كە چاوگ ڕوودانێك پیشاندەدات‌و بەكات نەبەستراوەتەوەو نەدراوەتە پاڵ كەسێك یان شتێك، بەڵام لێرەدا دەما‌نەوێت بزانین، كە چاوگ لە ڕووی دروستكردنییەوە چۆن پێكدێت؟ كاتێك سەیری ڕستەكان دەكەین، دەبینین چاوگ لە ڕووی دروستكردنییەوە بەم شێوەی خوارەوەییە:
یەكەم: لە ڕستەكانی بەشی (1) پەیڤەكانی (كوتان، هاتن، چاندن، دروون، پشكڤتن، خوێندن) هەموویان چاوگی سادەن، هەر یەكێك لەم چاوگانە ڕوودانێكی پیشانداوەو واتایكی سەربەخۆی هەیە، بەڵام ئەگەر لەڕووی دروستكردنەوە سەیریبكەین، دەبینین هەر یەكێكیان لە یەك وشەی واتادار پێكهاتووە. 
دووەم:
            ( ئـ )
	لە ڕستەكانی بەشی (2)ی بەشی (ئـ)دا وشەكانی (ڤەكرن، هەڵبژاردن، دەركەوتن، ڕاسپاردن، تێگهێشتن، سەركەڤتن، خوێندنەوە، قرتاندن، هەڵدانەوە) هەموویان چاوگی داڕێژراون، لە ڕووی دروستكردنیانەوە، هەر یەكێك لەم چاوگانە لە چاوگێكی سادە لەگەڵ پێشگرێك یان پاشگرێك یان لەكەڵ هەردووكیان پێكهاتووە، بەم شێوەی خوارەوە:
1. چاوگی (ڤەكرن) لە پێشگری (ڤە) لەگەڵ چاوگی سادە (كرن) پێكهاتووە.
2. چاوگی (هەڵبژاردن) لە پێشگری (هەڵ) لەگەڵ چاوگی سادە (بژاردن) پێكهاتووە.
3. چاوگی (ڕاسپاردن) لە پێشگری (ڕا) لەگەڵ چاوگی سادە 
(سپاردن) پێكهاتووە.
4. چاوگی (تێگهێشتن) لە پێشگری (تێ‌)  لەگەڵ چاوگی سادە (گهێشتن) پێكهاتووە.
5. چاوگی (خوێندنەوە) لە چاوگی سادە (خوێندن) لەگەڵ پاشگری (ەوە) پێكهاتووە.
6. چاوگی (قرتاندن) لە چاوگی سادە (قرتان) لەگەڵ پاشگری 
(اندن) پێكهاتووە. 
7. چاوگی (هەڵدانەوە) لە پێشگری (هەڵ)‌و چاوگی سادە (دان)و پاشگری (ەوە) پێكهاتووە.

                ( ب )
 	لە ڕستەكانی بەشی (2) ی بەشی (ب) دا پەیڤەكانی (ئاڤدان، یارمەتیدان، جوانكردن، گەرمبوون، دەستپێكردن، ڕاستكردنەوە، ل ناڤبردن، ڕێگەدان، بەهێزكردن) هەموویان چاوگی لێكدراون لە ڕووی دروست كردنیانەوە هەر یەكێكیان لە دوو وشەی واتادار یا پتر پێكهاتووە.
1. چاوگی (ئاڤدان) لە ناوی (ئاڤ ) لەگەڵ چاوگی سادە (دان) پێكهاتووە.
2. چاوگی (یارمەتیدان) لەناوی (یارمەتی) لەگەڵ چاوگی سادە (دان) پێكهاتووە.
3. چاوگی (جوانكردن) لە هاوەڵناوی (جوان) لەگەڵ چاوگی سادە (كردن) پێكهاتووە.
4. چاوگی (گەرمبوون) لە هاوەڵناوی (گەرم) لەگەڵ چاوگی سادە (بوون) پێكهاتووە.
5. چاوگی (دەستپێكرن) لە ناوی (دەست) لەگەڵ چاوگی داڕێژراوی (پێكرن) پێكهاتووە.
6. چاوگی (ڕاستكردنەوە) لە هاوەڵناوی (ڕاست) لەگەڵ چاوگی داڕێژراوی (كردنەوە) پێكهاتووە.
7. چاوگی (ل ناڤبرن) لە هاوەڵكاری داڕێژراوی (ل ناڤ ) لەگەڵ چاوگی سادە (برن) پێكهاتووە.
8. چاوگی (ڕێگەدان) لە ناوی داڕێژراوی (ڕێگە) لەگەڵ چاوگی سادەی(دان)پێكهاتووە.
9. چاوگی (بەهێزكردن) لە هاوەڵناوی داڕێژراوی (بەهێز) لەگەڵ چاوگی سادە (كردن) پێكهاتووە.

دەستور: 
	چاوگ لەڕووی پێكهاتنەوە دوو جۆرە:
1. چاوگی سادە:
 ئەو چاوگەیە، كە تەنیا لە یەك وشەی واتاداری سەربەخۆ پێكهاتووە. وەك: نووسین، خوێندن، نووستن، كێڵان، بوون، كردن
2. چاوگی ناسادە: دوو جۆرە.
ئـ- چاوگی داڕێژراو:
 ئەو چاوگەیە كە لە چاوگێكی سادە لەگەڵ پێشگرێك یان پاشگرێك یان لەگەڵ هەردووكیان پێكهاتووە:
1- (پێشگر+ چاوگی سادە) وەك: هەڵبوون، ڕاكردن، بەركەوتن، لێدان.
2- (چاوگی سادە+ پاشگر) وەك: خواردنەوە، خوێندنەوە، برژاندن، كوڵاندن.
3- (پێشگر+ چاوگی سادە+ پاشگر) وەك: هەڵبوونەوە، ڕاكردنەوە، لێدانەوە.
پێشگرەكان ئەمانەن: (هەڵ، دا، ڕا، پێ‌، تێ‌، دەر، بەر، ڤە، لێ‌).
پاشگرەكان ئەمانەن: (ەوە، اندن)
ب- چاوگی لێكدراو: 
لە چاوگێكی سادە یان داڕێژراو لەگەڵ وشەیەكی واتاداری تر یان پتر پێكهاتووە. بەم شێوەیە:
1- (ناو + چاوگی سادە) وەك: یارمەتیدان، دەستگرتن.
2- (هاوەڵناوی چۆنی+ چاوگی سادە) وەك: گەورەكردن، گەرمبوون، بڵندكردن، جێگیركردن.
3- (ناو+ چاوگی داڕێژراو) وەك: دەست هەڵگرتن، سەرهەڵبڕین.
4- (هاوەڵناوی چۆنی+ چاوگی داڕێژراو) وەك: ڕاستكردنەوە، چاكبوونەوە.
5- (ناوی داڕێژراو+ چاوگی سادە) وەك: بەخێوكردن.
6- (هاوەڵناوی داڕێژراو+ چاوگی سادە) وەك: بەهێزكردن، بەئاگاهاتن.
7- (هاوەڵكاری داڕێژراو+ چاوگی سادە) وەك: ل ناڤچوون، لەناوبردن.

ڕاهێنان (1)
	لەم چاوگە سادانەی خوارەوە چاوگی داڕێژراو و لێكدراو دروستبكە:
(چوون، كێشان، گرتن، خواردن، بڕین، هێنان)
ڕاهێنان (2)
	جیاوازی لە نێوان چاوگی داڕێژراو و لێكدراو چییە لە ڕووی دروستكردنەوە؟ بە نمونە ڕوونیبكەوە.
ڕاهێنان (3)
وەڵامی ڕاست هەڵبژێرە:
1. (برژاندن): چاوگێكی (سادەیە/ داڕێژراوە/ لێكدراوه/ هیچیان)
2. (ڕێزگرتن): پێكهاتووە لە 
             (ناو+چاوگی سادە / هاوەڵناو+چاوگی ساده / 
              هاوەڵكار+چاوگی ساده /  چاوگی سادە+ناو)
3. (گەرمبوون): پێكهاتووە لە 
(هاوەڵناو+پاشگر/ هاوەڵناو+چاوگی ساده/ ناو+ڕەگىكار/هاوەڵناو+ناو)
4. (لێنان): پێكهاتووە له
           ‌ (پێشگر+چاوگی ساده/    ناو+چاوگی ساده/
            پێشگر+چاوگی داڕێژراو/ چاوگی سادە+پاشگر)
5. (ڕاپەرین): پێكهاتووە لە 
(پێشگر+ناو/ ناو+ڕەگىكار/ هاوەڵكار+پاشگر/ پێشگر+چاوگی سادە)
6. كام لەم چاوگانە لە (پێشگر+چاوگی سادە+پاشگر) دروستبووە،
     (داشكاندن/ ڕاپەرین/ سەركەوتن/ كردنەوە)
