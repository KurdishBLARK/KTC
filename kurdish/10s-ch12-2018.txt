
كاری داڕژاو

1- قەڵاتی زیوی ڕووخاوە هەرەس بە تۆپی نەسیم
    سەپاهی لالەو گوڵ چادری لـــــــــــە جێ‌ هەڵدا          (حاجی قادری كۆیی)
2- مەمێ‌ ڕەبەن ئەڤیندار بـــــوو
    بەكۆ ڕابوو ژەهر تێكـــــــــــر
     ب كینا ڕەش ڤەكۆلا بـــــــوو
    ب زیندانا جزیرێ‌ كــــــــــــــر                            (سەبری بۆتانی)

3- كە دەركەوت مانگی دوو حەفتە سەراپەردەی درەخشان كرد
      بە عینوانی سەفا دەســـــــــتی جەفای لەو ناوەدا دەركرد               
                                                                                                                                        (بێخود)

4-خەرامان دەركەوت لە توێی پەردەوە  
    پەچەی لە ڕووخسار مانگ لابردەوە                          (مەولەوی)

5- با هۆزو باگەڕو بـــــــــــــــــــاران
     نەمان چوون و وەلات بــــــــەردان                       (سەبری بۆتانی)                   

6- لە ورشەی شەونم لە توێی پەردەوە
      نەرگس بە مەستی چـــاوی كردەوە                         (مەولەوی)

خستنەڕوو:

كاتێ‌ سەیری ئەم دێڕە هۆنراوانەی دەكەین، دەبینین وشەكانی (هەڵدا، ڕابوو، تێكر، ڤەكۆلا، دەركەوت، دەركرد، لابردەوە، بەردا، كردەوە) هەموو كارن، بەڵام ئەگەر بپرسین چ جۆرە كارێكن لە ڕووی دروستكردنەوە؟ دەڵێین كاری داڕژاون، چونكە هەریەكیان لە كارێكی سادە لەگەڵ پێشگرێك یان  پاشگرێك یا لەگەڵ هەردووكیان پێكهاتووە. بەم شێوەی خوارەوە.
1- لە دێڕە هۆنراوەی یەكەمدا كاری(هەڵدا) كارێكی داڕژاوە، لە پێشگری (هەڵ)و كاری سادەی (دا) پێكهاتووەو بووە بەكارێكی داڕژاو، لە چاوگی داڕژاوی (هەڵدان) وەرگیراوە.

2- لە دێڕە هۆنراوەی دووەمدا كاری (ڕابوو، تێكر، ڤەكۆلا) داڕژاون، كاری (ڕابوو) لەپێشگری (ڕا) و كاری سادەی (بوو) پێكهاتووەو بووە بە كارێكی داڕژاو، لە چاوگی داڕژاوی (ڕابوون) وەرگیراوە، كاری (تێكر= تێكرد) لەپێشگری (تێ‌)و كاری سادەی (كر- كرد) پێكهاتووەو بووە بە كارێكی داڕژاو، لەچاوگی داڕژاوی (تێكرن- تێكردن) وەرگیراوە.
كاری (ڤەكۆلا) لە پێشگری (ڤە) و كاری سادەی (كۆلا) پێكهاتووەو بووە بەكارێكی داڕژاو لە چاوگی داڕژاوی (ڤەكۆلان) وەرگیراوە.

3- لە دێڕە هۆنراوەی سێیەمدا كاری (دەركەوت- دەركرد) داڕژاون، كاری (دەركەوت) لەپێشگری (دەر) و كاری سادەی (كەوت) پێكهاتووەو بووە بە كارێكی داڕژاو، لە چاوگی داڕژاوی (دەركەوتن) وەرگیراوە

 كاری (دەركرد) لە پێشگری (دەر) وكاری سادەی (كرد) پێكهاتووە و بووە بەكاری داڕژاو، لە چاوگی داڕژاوی (دەركردن) وەرگیراوە.

4- لە دێڕە هۆنراوەی چوارەمدا كاری (دەركەوت، لابردەوە) داڕژاون كاری (دەركەوت) لە خاڵی پێشوو باسكراوە، كاری (لابردەوە) لە پێشگری (لا) و كاری سادەی (برد) و پاشگری (ەوە) پێكهاتووەو بووە بە كارێكی داڕژاو، لە چاوگی داڕژاوی (لابردنەوە) وەرگیراوە.

5- لە دێڕە هۆنراوەی پێنجەمدا كاری(بەردا) داڕژاوە، لە پێشگری (بەر) وكاری سادەی (دا) پێكهاتووەو بووە بە كارێكی داڕژاو، لە چاوگی داڕژاوی (بەردان) وەرگیراوە.

6- لە دێڕە هۆنراوەی شەشەمدا كاری (كردەوە) داڕژاوە، لە كاری سادەی (كرد) و پاشگری (ەوە) پێكهاتووەو بووە بە كارێكی داڕژاو، لە چاوگی داڕژاوی (كردنەوە) وەرگیراوە.


دەستور:  
كاری داڕژاو كارێكی ناسادەیە لە كارێكی سادەی واتا دارو پاشگرێك یان پێشگرێكی بێ‌ واتا پێك دێت یاخود لەگەڵ هەردووكیان پێك دێت بەم شێوەیەی خوارەوە:
یـەكـــەم: پێشگر + كاری سادە= كاری داڕێژراو  
هەڵواسی، هەڵبڕی، ڕاكرد، ڤەكر، داخست، ڤەمراند، ڕاپەڕی، وەرگرت، پێكرد، ڕونیشت، هەڵدا، داچوو، ڕویكر.
تێبینی (1):

ئەم پێشگرانە بەتەنیا هیچ واتایەكیان نییە، بەپێش كارە سادەكەی دەلكێن، بە گشتی پێشگری (هەڵ) واتای بەرزی و بۆسەرەوەیە، بەڵام پێشگری (دا) واتای بۆخوارەوەیە، پێشگری (ڕا) بزوتنەوەیەكی ڕاستەوخۆ پیشان دەدات، پێشگری (وەر) گەرانەوەو شوێن گۆڕین دەردەخات، پێشگری (ڕوو) بزوتنەوەیە بۆخوارەوە).
(هەڵدڕی، هەڵواسی، هەڵدا، هەڵستا، هەڵكرد، هەڵكەڤت) 
(داماڵی، دانیشت، داتەپی، دادڕی، دادا،........هتد)
(ڕابوو، ڕاماڵی، ڕاخست، ڕاكێشا، ڕاگرت،......هتد) 
(ڕووچوو- ڕوونیشت، ڕووهات....هتد)
(وەرگەڕا، وەرگرت، وەرپێچا، وەربادا،.......هتد) 

دووەم: پێشـگر + كاری سـادە + پاشـگر = كاری داڕژاو
هەڵدایەوە / داخستەوە / هەڵخستەوە 
هەڵ + دا + یــــ    + ەوە = هەڵدایەوە
دا+خست + ەوە (ڤە) = داخستەوە (داخستەڤە)
هەڵ + دا + ڤە = هەڵداڤە 
وەر+ پێچا + ڤە = هەڵداڤە 

سـێیەم: كاری ســادە + پاشــگری (ەوە) = كاری داڕژاو 
خوارد + ەوە = خواردەوە 
چوو + ەوە = چووەوە 
خوار + ەڤە = خوارەڤە 
چوو + ڤە = چووڤە 
هات + ەڤە = هاتەڤە 
یان : 
كاری سادە+ پاشگری «اند» كاری داڕژاو 
ڕژا+ اند= ڕژاند       سووتا+ اند= سووتاند 


تێبینی (2):
پاشگری (ەوە) دەچێتە سەركاری سادەو ئەم گۆڕانكارییانە نیشاندەدات دوای ئەوە كارێكی داڕژاویان لێ دروستدەكات. 

ئـ- واتای نوێ‌و جیاواز لە كارە سادەكە دروستدەكات وەك : 
 برد +ەوە= (بردەوە) لێرەدا بە واتای (سەركەوتن) دێت. 
كرد+ەوە= (كردەوە) لێرەدا بەواتای كردنەوەی دەرگاو پەنجەرەو شتێكی داخراو بەكاردێت. 
چوو+ ەوە= (چووەوە) لێرەدا بەواتای ڕەنگ چوونەوە بەكاردێت 
وەك :
1- یارییەكەیان بردەوە.
2- پەنجەرەكەی كردەوە.
3- ڕەنگی كراسەكەی بەهەتاو چووەوە.


ب- بەمانای دووبارەكردنەوەی كارێك دێت :
هات+ەوە= هاتەوە 
برد+ەوە = بردەوە 
كرد+ەوە= كردەوە 
1- هاوڕێ‌ جارێكی تر هاتەوە بۆ لامان.
2- ئەم جارەش كتێبەكەی بردەوە.
3- نوێژەكەم كردەوە .

پ- هەندێكجار، كە دەچێتە سەر كاری سادە، واتای كارەكە تەواو دەكات چونكە كاری وامان هەیە كە بەسادەیی واتایان نییە. وەك:
 حەسا + ی + ەوە = حەسایەوە 
  دۆزی + ی + ەوە = دۆزییەوە 
  كولا + ی + ەوە= كولایەوە 
  لەرا + ی + ەوە = لەرایەوە 

  1- میوانەكە لە لامان حەسایەوە 
  2- پارەكەمان دۆزییەوە 
  3- برینەكەی كولایەوە.






تێبینی (3): 
پاشگری (اند) لەبنچینەدا (اندن)ە، كە دەچیتە سەر قەدی یا، ڕەگی چاوگی سادەی تێنەپەڕ، چاوگی داڕژاوی تێپەڕ دروست دەكات وەك:
ڕژا + اندن= ڕژاندن 
مر + اندن= مراندن 
سەپا+ اندن= سەپاندن 
پاش لابردنی (ن) ی چاوگ كارێكی داڕژاوی تێپەڕ دروستدەبێت، چونكە وەكو دەزانین هەموو كارێك لە چاوگ وەردەگیرێت. 

تێبینی (4): 
لەبەرامبەر پاشگری (ەوە) لەشێوەی كرمانجی سەروودا پێشگری (ڤە) بەكاردێت (1) وەك :
ڤە + كر= ڤەكر 
ڤە + تەمراند = ڤەتەمراند 
ڤە + بوو = ڤەبوو
ڤە + ما = ڤەما  
ڤە + مراند = ڤەمراند 
ڤە + دا = ڤەدا 
1- دەرگەهـ ڤەكر   
2- ئاگر ڤەمراند   
3- پەنجەرك ڤەبوو  
 

 ڕاهێنان ( 1 ) 
لەم دێڕە هۆنراوانەی خوارەوەدا هەرچ كاری داڕژاو هەیە دەری بهێنەو لەڕووی پێكهاتنییەوە دەست نیشانیبكە. 
1- سبەینێ‌ كە جەنگ دامر كایەوە 
    ئاگری هەراو شـــەڕ كوژایەوە                                (بێكەس) 

2-  گەلێكم ڕۆژگاری تاڵ و شیرین ڕابوارد، ئــــاخۆ
      مەرگ مەودا دەدا دیسان ببینم سەردەمێكی تر               (هێمن)

3- چەند جار سوتام و هەڵگەڕامەوە 
      بووم بە خۆڵەمێش دامركامەوە                        (شێخ سەلام)

4- شەتری و تایێن سوورو زەر 
     باێی نەسیم لێدا ســــــــــەحەر
     ل وێ‌ جەبهەتێ‌ هژیان دووبەر
     لەورا سوكاری بێ‌ هشین                             (مەلای جزیری)

5- لێم گەڕێ‌ با دەربڕم ســــۆزی دەروون
     لێم گەڕێ‌ با هەڵوەرێنم ئەشكی ڕوون                         (هێمن)

6- گوڵێ‌ جانێ‌ تە ئەز برم 
     ژ دنیایێ‌ ئەز دەركرم                                 (سەبری بۆتانی)

ڕاهێنان ( 2 )

واتای ئەم پێشگرو پاشگرانەی خوارەوە دیاری بكەو لەگەڵ هەریەكێكیان دوو كاری داڕژاو دروستبكەو لە ڕستەدا بەكاری بێنە. 
(هەڵ ، وە، ڤە ، دا، ەوە)


ڕاهێنان ( 3 )

ئەم ڕستانە شیبكەوە: 
1- ئاڵای كوردستان لە هەموو شوێنێك دەشەكێتەوە.
2- لاپەڕەكانی گۆڤارەكەی لەسەرخۆ هەڵدەدایەوە.
3- وی پەنجەرەك ڤەكر.
4- مەمۆی ئاگرێ‌ نەورۆزێ‌ هەڵكر.
