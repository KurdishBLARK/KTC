چۆنی (نەرم) لەگەڵ پاشگری (ین) پێكهاتووە، ناوی (چاكە) لە هاوەڵناوی چۆنی (چاك) لەگەڵ پاشگری (ە) پێكهاتووە، ناوی (جوانی) لە هاوەڵناوی چۆنی(جوان) لەگەڵ پاشگری (ی) پێكهاتووە، واتە (هاوەڵناو+پاشگر).
9- لە ڕستەی نۆیەمدا وشەی (هاوكار، هاوڕێ‌) ناوی داڕێژراون و واتاو مەبەستێك پیشاندەدەن، لە ڕووی دروستكردنەوە هەریەكەیان لە پێشگری (هاو) لەگەڵ ناوی (كار) پێكاتووە، ناوی هاوڕێ‌ لە پێشگری (هاو) لەگەڵ ناوی (ڕێ‌) پێكهاتووە، واتا (پێشگر + ناو).
	
	هەروەها لە ڕستەكانی بەشی دووەم (ب) دەبینین:
1- لە ڕستەی یەكەمدا وشەی(دۆتمام، گوڵاڤ، گوڵەباخ) ناوی (لێكدراون) و واتاو مەبەستێك پیشاندەدەن، لە ڕووی دروستكردنەوە هەریەكەیان بەم شێوەیە پێكهاتووە، ناوی (دۆتمام) لە دوو ناو پێكهاتووە، (ناو+ناو)، (دۆت+مام) ناوی (گوڵاڤ) لە دوو ناو پێكهاتووە (گوڵ+ئاڤ) واتە(ناو+ناو)، ناوی (گوڵەباخ) لە دوو ناو لەگەڵ ئامرازی لێكدەری (ە)، كە كەوتۆتە نێوانیان پێكهاتووە، (گوڵ+ە+باخ)، (ناو+ە+ناو).
2- لە ڕستەی دووەمدا وشەی (نەوزاد) ناوێكی لێكدراوە و واتا و مەبەستێك پیشاندەدات، لە هاوەڵناوی (نەو) لەگەڵ ناوی (زاد) پێكهاتووە، واتە (هاوەڵناو + ناو).
3- لە ڕستەی سێیەمدا وشەی (گردەسۆر، كێوەڕەش )، ناوی (لێكدراون) و واتا و مەبەستێك پیشاندەدەن، ناوی (گردەسۆر) لە ناوی (گرد) لەگەڵ ئامرازی لێكداری (ە) و هاوەڵناوی (سۆر) پێكهاتووە، (ناو+ە+هاوەڵناو) ناوی (كێوەڕەش) لە ناوی (كێو) و ئامرازی لێكداری (ە) و هاوەڵناوی (ڕەش)  پێكهاتووە، (ناو+ە+هاوەڵناو).
4- لە ڕستەی چوارەمدا وشەی (ڕەشماڵ، ڕەشمار) ناوی لێكدراون واتاو مەبەستێك پیشاندەدەن، ناوی (ڕەشماڵ) لە هاوەڵناوی چۆنی (ڕەش) لەگەڵ ناوی (ماڵ) پێكهاتووە، ناوی (ڕەشمار) لە هاوەڵناوی چۆنی (ڕەش) لەگەڵ ناوی(مار) پێكهاتووە، واتە(هاوەڵناوی چۆنی+ناو).
5- لە ڕستەی پێنجەمدا وشەی (دڵخواز، سەرپۆش) ناوی لێكدراون و واتاو مەبەستێك پیشاندەدەن، ناوی (دڵخواز ) لە ناوی (دڵ) لەگەڵ ڕەگىكار (خواز) پێكهاتووە، ناوی( سەرپۆش) لە ناوی (سەر) لەگەڵ ڕەگىكار (پۆش) پێكهاتووە، واتە(ناو+ ڕەگىكار).
6- لە ڕستەی شەشەمدا وشەی (شۆڕەسوار) ناوێكی لێكدراوە و واتاو مەبەستێك پیشاندەدات، لە هاوەڵناوی (شۆڕ) و ئامرازی لێكدەری (ە) و ناوی (سوار) پێكهاتووە، (هاوەڵناو +ە+ناو).
7- لە ڕستەی حەوتەمدا وشەی (ژێرپیاڵە) ناوێكی لێكدراوەو واتاو مەبەستێك پیشاندەدات، لە هاوەڵكاری (ژێر ) لەگەڵ ناوی (پیاڵە) پێكهاتووە، واتە ( هاوەڵكار + ناو ).









دەستور
     ناو لەڕووی پێكهاتنەوە دەكرێت بە دوو جۆر: 
یەكەم: 
ناوی سادە : ناوێكە لە وشەیەكی واتادار پێكهاتووە واتاو مەبەستێكی دیاریكراوی هەیە وەك ( كەو، بەرد، هەور، دەست...هتد) 
دووەم:
 ناوی ناسادە: دەكرێت بە دوو جۆر: 
ئـ - ناوی داڕێژراو: ناوێكە لە وشەیەكی واتادار لەگەڵ پاشگرێك یان لەگەڵ پێشگرێك پێكهاتووە بەم شێوەیە : 
1- ( ناو + پاشگر ) وەك ( كوردستان، گوڵدان، شەرمین، جێگە) 
2- ( ڕەگىكار + پاشگر ) وەك (زانا، ناڵە، نڤیسەر، فڕگە) 
3- ( هاوەڵناو + پاشگر ) وەك (نەرمین، چاكە، جوانی) 
4- ( پێشگر + ناو ) وەك (هاوكار، هاوژین، هامڕاز) 
ب- ناوی لێكدراو : ناوێكە لە دوو وشەی واتادار یا پتر پێكهاتووە، بەم شێوەی خوارەوە: 
1- ( ناو + ناو ) وەكو ( دۆتمام ، سۆلاڤ ، ماستاو ) 
2- ( ناو + ە + ناو ) وەكو ( گوڵەباغ ، گوڵەگەنم ) 
3- ( ناو + ە + هاوڵناو ) وەكو ( گردەسۆر ، كێوەڕەش ، كێلەشین ) 
4- ( هاوڵناو + ناو ) وەكو ( ڕەشماڵ ، ڕەشمار ، سۆرگوڵ ، نەوزاد ) 
5- ( ناو + ڕەگی كار ) وەكو ( دڵخواز ، سەرپۆش ) 
6- ( هاوەڵكار + ناو ) وەكو ( ژێرپیاڵە ، ژێرزەمین ) 
ڕاهینانی (1)
	بۆ هەریەكێك لەم پاشگرانەی خوارەوە ، دوو ناوی داڕێژراو بهێنە و لە ڕستەدا بەكاریان بهێنە : ( ایی، ین، ەڵان، گار ) 

ڕاهێنان (2)
	پێنج ناوی لێكدراو بهێنە بە مەرجێك سێیان لە هاوەڵكارو ناوێك پێكهاتبێت و یەكێك لە دوو ناو پێكهاتبێت و ئەوی تر لە ناوێك و ڕەگىكار پێكهاتبێت . 

ڕاهێنان (3)
	وەڵامی ڕاست هەڵبژێرە
1. ناوی (زێڕین) پێكهاتووە لە
  (ڕەگی كار+پاشگر / ناو+پاشگر / هاوەڵناو+پاشگر / پێشگر+ناو+پاشگر)
2. (خێرخواز) پێكهاتووە لە
  (هاوەڵناو+ ڕەگی كار / ناو+ڕەگی كار/ ناو+ناو/ هاوەڵناو+ناو+پاشگر)
3. (خوێنەر) پێكهاتووە لە
  (ناو+ڕەگی كار /ڕەگی كار+پاشگر / پێشگر+ناو / هاوەڵكار+پاشگر)
4. (دڵسۆز) ناوێكی لێكدراوە، پێكهاتووە لە
  (هاوەڵناو+ناو / ناو+ ناو /  پێشگر+ ناو / ناو+ پاشگر)
5. (گەڕۆك)
(ڕەگىكار+پاشگر/ ناو+پاشگر/ هاوەڵناو+ڕەگىكار/ پێشگر+ڕەگی كار+پاشگر)
6. (قورئانخوێن) پێكهاتووە لە
(ناو+ڕەگىكار/ پێشگر+ڕەگىكار+پاشگر/ هاوەڵناو+ڕەگىكار/ هاوەلكار+ڕەگىكار)
7. (ڕۆژگار) پێكهاتووە لە
  (ناو+ڕەگىكار/ ناو+هاوەلناو/ ناو+پاشگر/ هاوەلكار+پاشگر)
8. ناوی (نەورۆز) بە چ شێوازێك دروستبووە
  (هاوەڵناو+ناو/ ناو+ناو/ هاوەڵناو+پاشگر/ هاوەڵناو+هاوەڵناو)
9. كام لەم ناوانە لێكدراو نییە.
