
دەمەكانی كاری ڕابردوو
1. دیار چوو بۆ قوتابخانە.
2. تابلۆیەكەم بینیوە.
3. بانو نانەكەی خواردبوو.
4. شادیار وێنەكەی دەكێشا.
5. بریا شكۆ بەیەكەم دەرچووبا.     

خستنەڕوو:
	ئەگەر سەیری ئەم ڕستانە بكەین، دەبینین وشەكانی ( چوو، بینیوە،  خواردبوو، دەكێشا، دەرچووبا)، هەموویان كاری ڕابردوون، هەر یەكەیان كردەوەیەكی پیشانداوەو دراوەتە پاڵ كەسێك یان شتێك، لە كاتی ڕابردوودا ڕوویداوە، واتە پێش كاتی ئاخاڤتن كارەكە ڕوویداوەو تەواویش بووە، بەڵام ئەگەر سەیری كاتی ڕوودانی كارە ڕابردووەكان بكەین دەبینین لە یەك جیاوازن.

دەستور: 
	كاری ڕابردوو: ئەو كارەیە كردەوەیەك پیشاندەدات كە لە ڕابردوودا ڕوویداوەو دراوەتە پاڵ كەسێك یان شتێك، كاری ڕابردووی ڕاگەیاندن كاتی جیاوازی هەیە:



1. كاری ڕابردووی نزیك: لەڕووی دروستكردنییەوە لە چاوگ دروستدەكرێت بە لابردنی پیتی (ن)ی چاوگ، وەك:

چاوگ       كاری ڕابردووی نزیك 
هەڵكشان              هەڵكشا
هێنان                  هێنا
نووسین               نووسی
دانیشتن               دانیشت
خوێندن               خوێند
چوون                 چوو
گەڕانەوە              گەڕایەوە1

2. كاری ڕابردووی تەواو: بەم شێوەیە دروستدەكرێت:
(كاری ڕابردووی نزیك+ وە) وەك:
       (قەدی چاوگ + وە)
هێنا + وە = هێناوە
نووسی + وە = نووسیوە
چوو + وە = چووە2





بەڵام ئەگەر كاری ڕابردووی نزیك كۆتایی بە پیتی نەبزوێنی (ت،د) هاتبوو، پیتی بزوێنی (و) دەخرێتە نێوان كاری ڕابردووی نزیك‌و نیشانەی (وە) وەك:
دانیشت + و + وە = دانیشتووە
خوێند+ و + وە = خوێندووە
دەكرێت لەجیاتی نیشانەی (وە) نیشانەی (یە) بەكار بێت وەك:
هێنا + یە = هێنایە
نووسی + یە = نووسییە.
	بەڵام دیسان ئەگەر كاری ڕابوردووی نزیك كۆتایی بە پیتی نەبزوێنی (ت، د) هاتبوو، پیتی بزوێنی (ی) دەخرێتە نێوانیان وەك:
دانیشت + ی + یە = دانیشتییە
خوێند + ی + یە = خوێندییە
	لە هەندێك ناوچەی كرمانجی سەروو بەم شێوەیە دروستدەكرێت:
یا + ڕابردووی نزیك + ی             بۆ ڕەگەزی مێ
یێ‌ + ڕابردووی نزیك + ی            بۆ ڕەگەزی نێر
وەك: یا + هینا + ی = یا هینای
      یێ‌ + چاند + ی = یێ‌ چاندی
تێبینی:  ئەم چاوگانەی كە پاشگری (ەوە ) یان پێوەیە بەم جۆرە ڕابردووی تەواویان لێدروستدەكرێت.
ئـ- لابردنی نونی چاوگ.
ب- دانانی نیشانەی (وە) لە شوێنی (ن)ی لابراو.
پ- دانانی ناوبەندی (ت) لە نێوان بزوێنەكان
خوێندنەوە            خوێند + ووە+ ت + ەوە = خوێندووەتەوە
 دۆزینەوە             دۆزی + وە + ت + ەوە = دۆزیوەتەوە
3. كاری ڕابردووی بەردەوام بەم شێوەیە دروستدەكرێت:
(دە + ڕابردووی نزیك)
وەك:   دە + هێنا = دەهێنا
        دە + نووسی = دەنووسی
        دە + وەرگرت = وەردەگرت
        دە + دروستكرد = دروستدەكرد
	لە كرمانجی سەروودا لە جیاتی نیشانەی (دە)، نیشانەی (د) بەكاردێت وەك:
  د + هێنا = دهێنا
  د + چوو = دچوو
  د + وەگرت = وەردگرت.
4. كاری ڕابردووی دوور بەم شێوەیە دروستدەكرێت:
كاری ڕابردووی نزیك + بوو:
هێنا + بوو = هێنابوو
نووسی + بوو = نووسیبوو
چوو + بوو = چووبوو
وەرگرت + بوو = وەرگرتبوو
تێبینی:
  1. كاری ڕابردوو بە ئامرازی (نە) دەكرێت بە نەرێ‌.
  2. كاری ڕابردووی نزیكی دانانی بەم شێوەیە دروستدەكرێت.
  (كاری ڕابردووی نزیك + با) وەك:
 هات+ با = هاتبا                    خوێند+ با = خوێندبا

ڕاهێنان (1)
لەم چاوگانەی خوارەوەدا كاری ڕابردووی نزیك‌و دوور دروستبكە:
( پێشكەوتن، نڤیسین، بردن، ئاڤدان، داخستن )

ڕاهێنان (2)
وەڵامی ڕاست هەڵبژێرە:
1. قەد لە چاوگ وەردەگیرێت بە لابردنی....
    (ئەلف/ نون/ ئەلف و نون)
2. ئەگەر (ن ) لە چاوگ لابدەین چ كارێكی لێ دروستدەكرێت......
   (داخوازی/ ڕابردووی نزیك/ ڕانەبردوو)
3. پرژە وانەكەی خوێندبوو.  (خوێندبوو) كارێكی ڕابردووی.....
    (دوور/ نزیك/ تەواو/ بەردەوام)
4. كاری ڕابردووی دوور بەم شێوەیە دروستدەبێت.....
(ڕابردووی نزیك+ ووه/ دە+ ڕابردووی نزیك/ ڕابردووی نزیك+ بوو)
5. كاری ڕابردووی نزیكی دانانی بەم شێوەیە دروستدەبێت.....
 (ڕابردووی نزیك+ با/ دە+ ڕابردووی نزیك/ ڕابردووی نزیك+ بوو)
 
ڕاهێنان (3)
	لەم چاوگانەی خوارەوەدا كاری ڕابردووی بەردەوامی ڕاگەیاندن‌و كاری ڕابردووی نزیكی دانانی دروستبكەو لە ڕستەدا بەكاریان بهێنە.
(سەركەوتن، هەڵكێشان، خوێندن، یارمەتیدان، فرۆشتن )

ڕاهێنان (4)
	لەم ڕستەی خوارەوەدا:
دوێنی پەخشان ڕۆژنامەی دەخوێندەوە.
1. وشەی (دوێنێ‌) چییەو ئەركی چییە؟
2. وشەی (ڕۆژنامە) شیبكەوە.
3. كاری (دەخوێندەوە) كام جۆری كاری ڕابردووە؟ چۆن دروستكراوە؟
4. بكەری ڕستەكە دیاربكە.
5. لە كاری (دەخوێندەوە) كاری ڕابردووی تەواو دروستبكە.
6. لە كاری  (دەخوێندەوە) كاری ڕابردووی دوور دروستبكە.
7. لەكاری (دەخوێندەوە) كاری ڕابردووی نزیكی دانانی دروستبكەو لەڕستەدا بەكاری بهێنە.

  
 


ڕاهێنان (3)
	وەڵامی ڕاست هەڵبژێرە:
1. (بنووسن،  بكێڵن) كاری داخوازییە، (ن) جێناوی لكاوە ئەركی :
      (بكەر /  بەركار /  دیارخەری ناو)
2. كاری داخوازی نەرێ دەكرێت بە :
      (مه  ‌/   نا  /   نی)
3. كاری داخوازی لە چاوگی خواردن :
     (خوارد   /  بخۆ   /  بخۆم)
4. كاری داخوازی (بڕۆ، بنوو) وەرگیراوە لە چاوگی :
     (خەوتن و نووستن  /  ڕۆیشتن و نووستن  /  نواندن و ڕواندن)
5. (بكرە، بخوێنە) كاری داخوازییە، (ە) جێناوی لكاوە بۆ كەسی :
     (دووەمی كۆ  /  دووەمی تاك  /  سێیەمی تاك)  

ڕاهێنان (4)
	ئەم ڕستانەی خوارەوە شیبكەوە:
1. لە جەژنی نەورۆزدا جلی جوان لەبەربكەن.
2. ‌‌‌‌دیارییەكە وەرگرن.
