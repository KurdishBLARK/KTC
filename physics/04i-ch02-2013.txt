سووڕەكانى تەزووى نۆرەكى
Alternating current circuts

(1 - 2) پێشەكى :-
   لەوانەیە بپرسین، بۆچی تەزووى نۆرەكى ( a.c ) لە جیهاندا بە باشتر دادەنرێت و بڵاوتر بەكاردەهێنرێت لە چاو تەزووى نەگۆردا ( d.c ) ؟
  وەڵامى ئەم پرسیارە ئەوەیە كە ڤۆڵتیەى نۆرەكى  كاتێك لە سەرچاوەوە وەردەگیرێت دەتوانرێت بۆ ڤۆڵتیەكى  بەرزتر یان نزمتر بگۆڕێت. ئەویش بە بەكارهێنانى ترانسفۆرمەرە. ئەمەش گرینگیەكى زۆرى هەیەبۆ گواستنەوەى وزەى كارەبا بە چوستیەكى زۆر لە شوێنى بەرهەمهێنانى بۆ شوێنەكانى بەكارهێنانى، ئەم گرینگیەش لەبەر كەمى بە فیڕۆچوونى وزەیە لە بەرگرى هێڵەكانى گواستنەوەدا. هەروەك لەبەشەكەى پێشتر باسكرا. وە لەڕووى كردارەوە بەرهەم هێنانى بڕى زۆر گەورە لە توانا بەهۆی تەزووى نەگۆڕ شتێكى ئاسان نیە بەڵكو تووشى زۆر گرفت و سەختى دەبین.
لەبەر ئەوەى تەزووى نۆرەكى گرینگیەكى تەكنیكى زۆرى هەیە بۆیە لەم بەشەدا هەوڵ دەدەین كە هەندێك زانیارى سووڕەكانى تەزووى نەگۆڕ وەربگرین.
ئێمە دەزانین، كاتێك كۆیلى دینەمۆى تەزووى نۆرەكى بەخێراییەكى نەگۆڕ لەناو لێشاویكى موگناتیسى چۆنیەك دەسووڕێتەوە، پاڵهێزێكى كارەبایی نۆرەكى پەیدادەكات، بڕى ئەم پاڵهێزە لە سفرەوە بۆ زۆرترین بڕى ( V max ) زیاد دەكات، ئینجا دەبێتەوە سفر، ئەمە بە ئاڕاستەك، پاشان ئاڕاستەكەى پێچەوانە دەبێتەوە، دیسان لە سفرەوە بۆ زۆرترین بڕ ئینجا بۆ سفر دەگۆڕێت. وە بڕى ئەو پاڵهێزە گۆڕاوە لەهەر ساتێكدا ( Vin ) لەگەڵ گۆڕانى شوێنى كۆیلى دینەمۆكە دەگۆڕێت بە پێی ئەم هاوكێشەیەى خوارەوە. 


گۆشەى ( Ө ) گۆشە لادانى كۆیلى دینەمۆ دەنوێنێت، ئەمەش ئەو گۆشەیە كە دەكەوێتە نێوان ئاڕاستەى چڕی لێشاوى موگناتیسى و هێلى ستوونى لەسەر رووتەختى كۆیلەكە. كاتێك دوو جەمسەرەكەى  دینەمۆ بە سووڕێكى دەرەكى دەبەسترێت كە تەنها سیفەتى بەرگرى هەبێت، ئەوا بڕى ئەو تەزووەى ئەم ڤۆڵتیە نۆرەكیە بە سووڕەكەى دەدات بە هەمان شێوەى ڤوڵتیە دانراوەكە دەگۆڕێت  
                                                لە هەر لەرەیەكدا.
بڕى ئەم تەزووە نۆرەكیەش لەهەر ساتێكدا بە پێی ئەم هاوكێشەیە دەبێت.


لە وێنەى   ( 2 – 1a )دینەمۆیەكى سادەى تەزووى نۆرەكى پیشاندراوە، لە وێنەى  ( 2 – 1b ) چەماوەى ڤۆڵتیەى نۆرەكى پیشان دەدات كە بڕەكەى وەك گۆڕانى نەخشەى تەژێ  ( ساین ، sin  ) دەگۆڕێت 

























(2 - 2)  :- نواندى بڕە نۆرەكیەكان بە تیراساى رەوگەیی .
ئەگەر دوو بڕى نۆرەكى گۆڕاو، گەورەترین نرخیان لە هەمان ساتدا ڕوونەدات ئەوە دەڵێن لە نێوانیاندا جیاوازى ڕەوگە ( Phase difference  ) هەیە.
  بۆ ئەوەى بە باشى لە جیاوازى ڕەوگە تێبگەین و شى بكەینەوە، ئەوا ئەو بڕانەى بە شێوەى ساین (تەژێ) دەگۆڕێن بە تیراساى خولخۆر ( تیراسای رەوگەیی ) (  Phasors) دەنوێنین.
 لە وێنەى ( 2 – 2a  ) دا، op هێڵەكە بە دەورى (o  ) دەسووڕێتەوە بە خێراییەكى نەگۆڕ بە ئاڕستەى پێچەوانەى جوولەى میلى كاتژمێر، لەم ساتەدا op ، گۆشەى Ө لەگەڵ ( 0A ) دروست دەكات. بەم هێلە خولاوە دەگوترێ تیراسای ڕەوگەیى ( Phasor )، بەكاردێت بۆ نواندى بڕە نۆرەكیە گۆڕاوەكان، وەك ڤۆڵتیەو تەزووى نۆرەكى و پیشاندانى گۆشەى جیاوازى ڕەوگە لە نێوانیاندا. كاتێك    0 = Ө ئەوا op لەسەر 0A  جووت دەبێت بەگۆشەى ( Ө ) دەگوترێت گۆشەى رەوگە.  جێكەوتى تیراسای رەوگەیى op  لەسەر تەوەرى ( y ) دەبێتە ( ON )، وە لەسەر تەوەرەى ( x )  دەبێتە OM


ئەگەر درێژى تیراسای رەوگەیی ( op ) زۆرترین بڕە نۆرەكیەكە بنوێنێت ئەوا پێكنەرى ( ON ) بڕە ساتیەكەى دەنوێنێ.
دەتوانین هەریەك لە ڤۆڵتیە و تەزووى نۆرەكى بە تیراساى خولخۆر بنوێنین كە بە هەمان لەرەلەرەى ڤۆڵتیەكە بەدەورى خاڵى دەست پێكردنى دەسووڕێتەوە. لە وێنەى ( 2-2b )دا پەیوەندى ڕەوگەیی لە نێوان ڤۆڵتیە و تەزوو ڕوون دەكاتەوە كە تیایدا تەزوو بە گۆشەى جیاوازى ڕەوگە Φ پێش ڤۆڵتیە كەوتووە، ئەم پەیوەندییە بە شێوەى تیراساى ڕەوگەیی وە بە شێوەى چەماوەى (V, I) پیشاندراوە.

پێشەكى : زرنگانەوە بە مانا گشتییەكەى دیاردەیەكى سروشتیە لە نیزامێكداڕوودەدات كاتێك ئەو نیزامە دەكەوێتە ژێر كاریگەرى سەرچاوەیەكى لەراوە، لەرەلەرەكەى یەكسان بێت بەلەرەلەرى سروشتى نیزامەكە .

(12-2) دیاردەى كارۆپەستان                           Piezo electric effect
هەندێ بەللورەى تایبەت هەن نەگەیەنەرن، وەك كوارتز و تۆرمالین وخوێى رۆشیل، سیفەتێكى كارۆمیكانیكى سەیریان هەیە،ئەگەر یەكێك لەم بەللورانە لە نێوان دوو تەبەقى كانزاى دابنرێت، ئینجا بەللورەكە بخرێتە ژێر فشارێكى پەستاوتن وێنە (2-19a)، ئەوا ناوچەیەكى بەللورەكە بە بارگەى مووجەب ناوچەكەى دى بەرامبەرى بە بارگەى سالیب بارگاوى دەبن ، لە ئەنجامدا جیاوازیەكى ئەركى كارەبایى لە نێوان ئەم دوو ناوچەیە پەیدا دەبێت.
ئەگەر ئێستاكە ئاڕاستەى فشارەكە پێچەوانە بكرێتەوە، واتە بەللورەكە بخرێتە ژێر فشارێكى راكێشان، وێنە (2-19b) .ئەوا دوو جەمسەرەكەى بارگەكان پێچەوانە دەبنەوە، لە ئەنجامدا ئەركەكەش پێچەوانە دەبێتەوە.


	











بەدیاردەى بەكارەبابوونى دوو رووى بەرامبەر لە بەللورەیەكدا لەكاتى پەستاوتن وراكێشانى بەللورەكە دەگوترێ دیاردەى كارۆپەستان .
بڕگەیەكى تایبەت لە خوێى رۆشیل وەك مایكرفۆن بەكاردەهێنرێت، بەمەرجێك بەجۆرێك دابنرێت كە شەپۆڵەكانى دەنگ فشارێكى پەستاوتن بخەنەسەرى . جیاوازى ئەرك لەم بارەدا لە نێوان
 (10 mv        100 mv) دا دەبێت ، هەمان بیرۆكە لە فۆنۆگراف بەكاردێت كاتێك دەرزى فۆنۆگرافەكە لەناو درزەكانى قەوانەكە لەرەلەردەكات.
 هەندێ مادەى سیرامیك،  سیفەتى بەللورەكانى كارۆپەستانیان هەیە دەتوانن هەزاران ڤۆڵت پەیدابكەن ، وەك چەغماغەیەك بۆ داگیرسانى گازەكان  بەكاردێت ، كاتێك بەللورەكە دەكوشرێت، ڤۆڵتیەكى بەرز پەیدادەبێت ، ئەمەش پریشكێكى كارەبایى پەیدادەكات بۆ سووتانى گازەكە .ئەم دیاردەیە دیاردەیەكى پێچەوانە بۆوەیە، كاتێك ئەم جۆرەبەللورانە دەخرێنە ناو بوارێكى كارەبایى نۆرەكى ئەوا یان دەچنەوەیەك یان دەكشێن بەگوێرەى ئاراستەى بوارەكە. كاتێك ڤۆڵتیەكى نۆرەكى دەخرێتە سەر دوو رووى دیاریكراوى بەرامبەر یەكى بەللورەى كوارتز ( لە نێوان دوو تەبەق دانرا بێت وێنە (20-2)ئەوا بەللورەكە دەچێتەوە یەك و دەكشێت ،بەهەمان لەرەلەرى ئەو ڤۆڵتیەى خراوەتە سەرى دەلەرزێت، ئەم سیفەتەش.بۆ كۆنترۆلكردنى لەرەلەرى سووڕەكانى لەرینەوەى رادیۆیى، لە سەعاتى كوارتزدا هەروەها بۆ دەستكەوتنى شەپۆڵە سەرو دەنگیەكان وەك لە دەزگاى سۆناردا بەكاردێت.





	




( 13-2) ئەمیتەرى تەلى گەرم 
كاتێك گلڤانۆمەتر لە سووڕێكى نۆرەكى دادەنرێت، ئەوا نیشاندەرەكەى لەبەر سیفەتى بارنەگۆرینى كۆیلى گلڤانۆمەترەكە ناتوانێت ئاڕاستەى لادانەكەى بگۆڕێت لەگەڵ گۆراِنى ئاڕاستەى تەزووەكە .بۆیە نیشاندەرى  گلڤانۆمەتر لەسەر سفر دەوەستێت،لەمەوە  دیاردەكەوێت كە ناتوانرێت پێوەرەكانى تەزووى نەگۆڕ وەك ئەمیتەر وڤۆلتمیتەر ڕاستەوخۆ لە سووڕەكانى نۆرەكیدا بەكاربهێنن. بۆ پێوانى تەزووى نۆرەكى پێویستە یەكەم جار بگۆڕدرێت بۆ تەزووى نەگۆڕ بە بەكارهێنانى راستكەرەوە (rectifier)ئینجا ئەمیتەرى كۆیل جوڵاوە بەكاربهێنرێت بۆ پێوانى تەزووەكە یان ڤۆلتیەكە .دەتوانرێت ڕاستەوخۆ تەزووى نۆرەكى بپێورێت بەبەكارهێنانى ئەمیتەرى تەلى گەرم كە بنچینەى ئیشكردنى بە كاریگەرى گەرمى تەزووەوە بەندە.
